📋 Задание 5: Таблица (собаки / коты)

Последовательность: dog dog dog cat cat dog dog cat cat dog dog cat cat ...

В каждой строке — 6 животных, разделитель — табуляция.



📌 Результат выводится в КОНСОЛЬ браузера (F12 → Console)
Пример для 6 строк: 
dog     dog     dog     cat     cat     dog
dog     dog     cat     cat     dog     dog
dog     cat     cat     dog     dog     dog
cat     cat     dog     dog     dog     cat
cat     cat     dog     dog     cat     cat
cat     cat     dog     dog     cat     cat
